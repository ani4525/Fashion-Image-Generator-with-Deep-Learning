# 👗 AI Fashion Generator using Stable Diffusion

## 📌 Overview
AI Fashion Generator is a deep learning-based application that generates high-quality clothing designs from text prompts. It uses Stable Diffusion to convert user input into realistic fashion images, demonstrating the power of Generative AI in the fashion industry.

---

## 🎯 Objectives
- To generate fashion images using AI
- To implement text-to-image generation
- To explore Generative AI in real-world applications
- To build an interactive user interface for design creation

---

## 🚀 Features
- Text-to-image fashion generation
- Clean and interactive UI
- Real-time image generation
- AI-based model integration
- Custom prompt input

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Framework:** Flask  
- **AI Model:** Stable Diffusion  
- **Libraries:** PyTorch, Transformers, Diffusers  
- **Frontend:** HTML, CSS, JavaScript
